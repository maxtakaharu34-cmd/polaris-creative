# AUTO KING — オートキング

中古車販売・新車・買取・整備の総合カーディーラー HP。
Valenti Japan 風のダーク×レッドの大胆なデザイン。

## Tech Stack

- Vite + React 19 + TypeScript
- Tailwind CSS v4
- Framer Motion
- Oswald + Noto Sans JP

## 開発

```bash
npm install
npm run dev      # http://localhost:5176
npm run build    # 本番ビルド → dist/
```

## デプロイ

`main` への push で **GitHub Pages へ自動デプロイ**。
